# BR-028 — Несколько писем при восстановлении пароля

## Описание
Если несколько раз нажать кнопку "Восстановить пароль", приходит несколько писем с разными паролями.

## Шаги воспроизведения
1. Перейти по ссылке https://lm.skillbox.cc/qa_tester/module06/auth/index.html
2. Нажать "Забыли пароль?"
3. Ввести в поле E-mail: `skillbox@test.ru`
4. Несколько раз подряд нажать кнопку "Восстановить пароль"

## Фактический результат
- Приходят несколько писем с разными паролями. Если ввести пароль не из последнего письма, вход не происходит.

## Ожидаемый результат
- Должно приходить только одно письмо с последним актуальным паролем, либо каждый новый запрос должен инвалидировать предыдущие пароли.
